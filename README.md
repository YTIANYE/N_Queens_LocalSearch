# N_Queens_LocalSearch
（随机重爬）
N皇后(1000~ 1000000)
要求：
任意输入一个整数N, 103  £ N £ 106。
在普通个人计算机上，对输入N，3分钟内给出一个皇后不冲突的方案。
输出到磁盘。保存输出文件名为“N.txt”。N.txt用32位无符号整数的二进制存储，数值范围从0 ~ N-1；统一用大端（Big-Endian）。
提供独立的检验程序和代码，检验所输出文本文件中的皇后是否冲突。

测试结果
    200000个皇后的问题在普通PC上基本可以在1分钟内解决。
